# bloc_template

Production-ready Flutter scaffold generated by the Flutter Scaffolding Wizard.

## What's inside

- Opinionated theme with Material 3
- Onboarding presentation starter
- Routing scaffold using `go_router`
- State management with `bloc`
- Custom backend setup with Dio
- Shared local and secure storage services
- `easy_localization` translations for `en` and `es`

## Creating A New App From This Template

When you create a new GitHub repository from this template, rename the app before generating native platform files.

```bash
flutter create --project-name my_app --org com.yourcompany .
```

Use:

- `--project-name my_app` for the Dart package name in `pubspec.yaml`
- `--org com.yourcompany` for the native package prefix

The final native app ID becomes similar to:

```text
com.yourcompany.my_app
```

After running `flutter create`, replace all template package references:

```bash
perl -pi -e 's/bloc_template/my_app/g' $(find lib test -name "*.dart") pubspec.yaml README.md AGENTS.md DESIGN.md SETUP.md
```

Then update:

- `lib/src/app.dart` app title
- `.env` `API_BASE_URL`
- `flutter_native_splash.yaml` color and splash image
- app icons and native display name if needed

Verify:

```bash
flutter pub get
flutter analyze
flutter test
```
